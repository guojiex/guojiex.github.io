你好，小乌龟！

在Python中有很多强大的预置模块，我们可以把他们运用到自己的程序中去。他们中的一些可以发送邮件，或者抓取网页。这章中我们将要见到的模块，允许我们创建一只乌龟（？）并且使用它来绘图。

小乌龟很萌，但是这一章的目标是教我们更多关于Python的知识，以及发展属于我们自己的计算思考的方向，或者如何像一个计算机科学家那样思考。本章中用到的Python语句，会在后面的章节进行更深入的讲解。
